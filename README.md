# Rust Puzzles

Frontend for [Simon Tatham's Portable Puzzle Collection](https://www.chiark.greenend.org.uk/~sgtatham/puzzles/) written in Rust.

Uses the following libraries:

- [winit](https://github.com/rust-windowing/winit) for windowing and event handling
- [pixels](https://github.com/parasyte/pixels) for window rendering
- [raqote](https://github.com/servo/raqote) for software drawing
